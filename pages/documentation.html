Contact Form

Update the action attribute on each contact form so it points to the service that will receive submissions (FormSubmit, Formspree, Netlify Forms, Zapier webhook, etc.).

<form id="contactForm" role="form" method="post" action="https://formsubmit.co/ajax/CHANGE_ME@example.com">
Replace the placeholder URL with the endpoint provided by your form service. The JavaScript in assets/js/main.js handles the AJAX request and shows a reminder if the placeholder is left unchanged.
When testing locally without replacing the placeholder, the form displays a warning instead of sending data so you do not lose submissions.

Reservation Form

The reservation form uses the same static submission flow. Point the action attribute at an endpoint that can capture booking requests.

<form class="reservation-form" id="reservationForm" action="https://formsubmit.co/ajax/CHANGE_ME@example.com" method="post">
Use custom fields in your provider (or Google Forms, Airtable, etc.) to mirror the form inputs so the submitted data is stored correctly.

Configuring Mailchimp

The newsletter form posts to the same configurable endpoint. Set the action attribute to the URL supplied by your email marketing tool.

<form id="subscription-form" role="form" method="post" action="https://formsubmit.co/ajax/CHANGE_ME@example.com">
For Mailchimp, create an embedded form (Audience → Signup forms → Embedded forms) and replace the template's newsletter form with the snippet Mailchimp generates, or use a compatible third-party service that accepts POST requests.

Plugin Settings

All plugins setting is in  assets/js/custom.js  file

  • Appear - Progress bars, CountTo
  • Text rotator - Changes to different text in same place. Mainly used in Header
  • Isotope - Portfolio
  • FlexSlider - Sliders
  • OWL Carousel - Team carousel, client logos carousel
  • Magnific-popup - Modal window, gallery, modal video
  • Video background - Header video background or any section with video
